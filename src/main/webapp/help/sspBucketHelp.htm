Manage SSP Bucket Help

Manage SSP Buckets allows an administrator to view, add, or delete SSP Buckets for a given user in the Sample Run-Time Environment.

Manage SSP Buckets Instructions
  1. Select the user from the drop down box whose SSP Bucket information you would like to manage.
  2. Click Submit to manage the SSP Buckets associated with the selected user.
  3. All current buckets for the selected user will be visible in the table.
  4. To delete a bucket, click the radio button next to the Bucket ID and click Delete.
  5. To add a bucket, click the Add button.
  6. Complete the Add Bucket fields with the correct values in the remaining steps. Only bucketID, requested, courseID, and scoID are required.
  7. Add a bucketID containing text or numeric values.
  8. Add a requested value containing only numeric values.
  9. Add a minimum value containing only even numeric values.
  10. Add a reducible value by choosing a value from the drop down box.
  11. Add a persistence value by choosing a value from the drop down box.
  12. Add a type value containg text or numeric values.
  13. Add a courseID containing text or numeric values.
  14. Add a scoID containing text or numeric values.
  15. Click Create Bucket to create the SSP Bucket.
