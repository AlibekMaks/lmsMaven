Manage Courses Help

Manage Courses allows an administrator to view and edit comments about any items contained in a course.

Manage Courses Instructions
  1. Choose the title of the course you wish to manage from the drop down box and click Submit.
  2. Choose the title of the item whose comments you wish to view or edit from the drop down box and click Submit.
  3. To add a comment, click the Add Comment button, enter the Comment and its optional Location in the Add Comments Window, and click Add Comment. Additional comments can be entered from the Add Comments Window or Close can be clicked to return to the View Comments page.
  4. To reset the comments visible in the text area, click the Reset Comments button.
  5. To store the comments visible in the text area, click the Submit button.
