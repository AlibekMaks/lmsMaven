Environment Variable

  • The Environment Variable will be set when the Sample Run-Time Environment is installed. If for any reason it would need manually reset, please follow these instructions.
    1. Right-Click My Computer
    2. Click the Advanced Tab
    3. Click the Environment Variables button
    4. Click the New button
    5. Set the Variable name to SCORM3ED_SRTE102_HOME
    6. Set the Variable value to <drive>:\<install_directory>
    7. Click Ok on the New Variable window
    8. Click Ok on the Environment Variables window
    9. Click Ok on the Systems Properties window
