System Requirements

  • Minimum System Requirements for operation of the Sample Run-Time Environment:
    • Windows XP or Windows Vista
    • 2 GHz Processor
    • 1 GB of RAM
    • 35 MB of free disk space

  • Recommended System Requirements for operation of the Sample Run-Time Environment:
    • Windows XP or Windows Vista
    • 4 GHz Processor
    • 2 GB of RAM
    • 35 MB of free disk space
