ODBC Connections

  • The ODBC Connections will be set when the Sample Run-Time Environment is installed. If for any reason they would need manually reset, please follow these instructions.
    1. Open the Windows "Control Panel"
    2. Double-click Administrative Tools
    3. Double-click Data Sources (ODBC)
    4. If you are an Administrator, choose the System DSN tab
    5. If you are a user, choose the User DSN tab
    6. Click Add
    7. Choose Microsoft Access Driver (*.mdb), and click Finish
    8. Enter SCORM3EDGlobalObjs102 for Data Source Name:
    9. Click Select and browse to <drive>:\<install_directory>\Sample_RTE\database\
    10. Select the GlobalObjectives.mdb database and click OK
    11. Repeat steps 6 through 10 using the Data Source Name SCORM3EDSRTE102 matched to the SampleRTE.mdb database, and the Data Source Name SCORM3EDSSP102 matched to the SSPManager.mdb database
    12. Click OK on the ODBC Data Source Administrator window
